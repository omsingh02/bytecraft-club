## What this is

A polished landing page for a student-run technical club. Activities, focus areas, upcoming events, member projects, core team, and join CTA.

## Sections

1. Hero with community stats
2. What we do — activity cards
3. Focus areas — domain pill tags
4. Upcoming events — event rows
5. Projects — open-source work
6. Core team — member cards
7. Join CTA
8. Footer

## Tech stack

- HTML
- CSS (custom properties, mobile-first, amber accent theme)
- No frameworks, no build tools

## Run locally

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```
