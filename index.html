What we do

Build. Learn. Ship.

Not a club that exists only on paper. We actually build things and help each other get better at it.

Technical sessions

Hands-on workshops on real topics — Git, Linux, web dev, databases, scripting. No death-by-slide-deck.

Project sprints

Short collaborative builds where teams work together on a shared project for 2–3 weeks.

Open-source contributions

We maintain student-built tools and encourage members to contribute to external FOSS projects.

Hackathon prep

Team formation, brainstorming, and build practice before campus and inter-college hackathons.

Peer code reviews

Submit your code, get feedback from peers. No judgment — just better code and better habits.

Community hangouts

Informal sessions to talk tech, share finds, and just spend time around other builders.
